crowkit

Portable second brain + Claude Code harness that travels across machines.

$ npx crowkit

🔍 Auth Preflight

Checks GitHub CLI, npm, Claude Code, and git config. Tells you exactly what's missing before you start.

📚 Karpathy Wiki

Three-layer knowledge base: raw sources you drop in, wiki pages the LLM maintains, and generated outputs.

☁️ Cross-Platform Sync

iCloud on Mac, local + git everywhere else. Skills, CLAUDE.md, and config travel with you.

🔐 Native Secret Storage

API keys in macOS Keychain, GNOME Keyring, or Windows Credential Manager. Never in plain text.

🤖 Skills: /next /ingest /lint

Built-in Claude Code skills for wiki operations. Ingest sources, health-check pages, review what's next.

⚡ One Command

Run npx crowkit on any machine. Auth preflight, wiki setup, skills, secrets — all handled.

The Karpathy Pattern

Inspired by Andrej Karpathy's LLM Wiki

📄

raw/

You own this

Articles, notes, screenshots, PDFs. Immutable inputs the LLM reads.

📖

pages/

LLM owns this

Wiki pages created, updated, and cross-linked by Claude Code.

📊

outputs/

LLM owns this

Reports, answers, and analysis generated from queries.
